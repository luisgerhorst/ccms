Fist you have to add these lines to your Apache's httpd.conf file.

LoadModule proxy_module modules/mod_proxy.so
ProxyRequests On
ProxyPass {{proxy}}/_users/ http://127.0.0.1:5984/_users/
ProxyPass {{proxy}}/_config/admins/ http://127.0.0.1:5984/_config/admins/
ProxyPass {{proxy}}/{{database}}/ http://127.0.0.1:5984/{{database}}/

Then restart Apache and make sure CouchDB is running.
