Read this attentively:

If you don't have CouchDB installed on your server, you find instructions here.

So CCMS can access your CouchDB database you'll have to a configure an Apache proxy, create a database and a CouchDB admin account. You can do all this using the shell script install.sh but you can also do it manually.

Execute this line in your shell, it will do nothing without your confirmation. You'll have to enter your root password.

sudo sh /path/to/ccms/install.sh
