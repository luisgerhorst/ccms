Apache 2

Add this code to Apache 2's apache2.conf.

LoadModule proxy_module /usr/lib/apache2/modules/mod_proxy.so
LoadModule proxy_http_module /usr/lib/apache2/modules/mod_proxy_http.so
ProxyRequests On
ProxyPass {{proxy}}/_session/ http://127.0.0.1:5984/_session/
ProxyPass {{proxy}}/_config/admins/ http://127.0.0.1:5984/_config/admins/
ProxyPass {{proxy}}/{{database}}/ http://127.0.0.1:5984/{{database}}/

XAMPP

Add this code to XAMPP's httpd.conf.

ProxyRequests On
ProxyPass {{proxy}}/_session/ http://127.0.0.1:5984/_session/
ProxyPass {{proxy}}/_config/admins/ http://127.0.0.1:5984/_config/admins/
ProxyPass {{proxy}}/{{database}}/ http://127.0.0.1:5984/{{database}}/

Make sure CouchDB is running and restart your web server.
